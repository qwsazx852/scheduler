# 如何重新啟動 StockGuard

如果您關閉了電腦或終端機，請按照以下步驟重新開啟程式：

1. **開啟終端機 (Terminal)**
   - 按下 `Command + Spacer`，輸入 `Terminal` 並開啟。

2. **進入程式資料夾**
   複製並貼上以下指令，然後按 `Enter`：
   ```bash
   cd /Users/jun/.gemini/antigravity/scratch/stock_guard
   ```

3. **啟動程式**
   輸入以下指令並按 `Enter`：
   ```bash
   npm run dev
   ```

4. **開啟網頁**
   - 看到類似 ` Local:   http://localhost:5173/ ` 的訊息後。
   - 按住 `Command` 鍵並點擊該網址，或是在瀏覽器輸入 `http://localhost:5173`。
